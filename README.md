# Ethereum Classic Transaction Monitor Bot

Этот бот отслеживает транзакции по 10 заданным Ethereum Classic (ETC) адресам и уведомляет вас, если сумма перевода превышает 10,000 монет.

## 🚀 Как запустить

### 1. Подготовка
- Создайте бота в Telegram через [@BotFather](https://t.me/BotFather) и получите `API TOKEN`.
- Узнайте свой `CHAT_ID` (можно через [@userinfobot](https://t.me/userinfobot)).
- Получите 10 ETC-адресов, которые хотите отслеживать.

### 2. Настройка среды
1. Снимите копию с `.env.template` и назовите ее `.env`.
2. В файле `.env` укажите:
   - `TELEGRAM_BOT_TOKEN`: полученный токен.
   - `MONITORED_ADDRESSES`: список из 10 адресов через запятую.
   - `ADMIN_CHAT_ID`: ваш Telegram ID.
   - (Опционально) `THRESHOLD_ETC`: порог по монетам (по умолчанию 10000).

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск бота
```bash
python bot.py
```

## 🛠 Технологический стек
- **Python 3.8+**
- **aiogram**: Асинхронная библиотека для Telegram-ботов.
- **web3.py**: Для взаимодействия с блокчейном Ethereum Classic.
- **python-dotenv**: Для управления конфигурацией.

## 📝 Заметки
- Бот каждые 10 секунд проверяет новые блоки в сети ETC.
- Бот уведомляет как об исходящих, так и о входящих транзакциях для указанных адресов.
- Хэш транзакции будет кликабельным ссылкой на блок-эксплорер.
